When in the myclient folder run the client.py file

This will prompt you for commands, its has all the valid commands from the specification apart from list 

login url - login function prompts further information
logout - logout function
post - post story function, prompts further information 
news -cat=x -reg=x -date=x - get stories function (no id as no directory service implemented)
delete key - delete story with id=key function
exit - exits the loop and ends program execution

The loop will continue running until you type 'exit' and this ends execution
